Test statistics

This page gives the links to thetext based pages with test results we gather each day.

We gather the results forall pages and for the clean builds. With clean builds we mean all builds that don't have compile errors. This is mainly done because builds that do have compile errors mostly have all tests failing which can give an incorrect idea of the status of thetest results.
